# Personal Portfolio

Modern multilingual portfolio built with **Next.js 15** and **motion/react** animations.

**Live site:** https://igorator.site

## Tech Stack
- **Next.js 15** (App Router, RSC)
- **TypeScript**
- **CSS modules**
- **motion/react** (Framer Motion v11 API)
- **next-intl** 
- **JSON-based data** (merged via server utilities)
